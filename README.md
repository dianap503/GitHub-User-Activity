# GitHub Activity CLI

Un utilitar simplu pentru linia de comandă (CLI) care preia și afișează activitatea recentă a unui utilizator de GitHub direct în terminal.

## Funcționalități
- Preluarea datelor prin API-ul GitHub.
- Filtrarea activității (Push, Issues, Watch).
- Formatarea lizibilă a output-ului.

## Instalare
1. Clonează acest repository (sau descarcă fișierul `github-activity`).
2. Asigură-te că ai permisiuni de execuție:
   ```bash
   chmod +x github-activity\
3. Introdu comanda:
   './github-activity <username>'
